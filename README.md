# SweetPlayerMarket

Minecraft 玩家全球市场插件。

## 简介

SweetPlayerMarket 是一个支持跨服、支持数据库的玩家全球市场插件。

玩家可以在全球市场以**出售**或**收购**的方式，上架自己的物品，让其他玩家可以自由购买或出售他们的物品。

同时，你也可以限制玩家的上架行为，让某些物品禁止上架，或者只能使用某种货币上架，还可以收取上架手续费。

利用数据库丰富的查询参数，本插件允许你以丰富的条件来排序商品列表。例如，按 `创建时间`、`到期时间`、`价格`、`数量` 其中的一个数据表列名进行 `升序 (从小到大)` 或 `降序 (从大到小)` 的方式进行排序。

本插件依然沿用 [PluginBase](http://plugins.mcio.dev/elopers/base/gui-config) 的高可自定义界面引擎，允许你让商品列表以**任何奇形怪状的形式**显示在这最大 `6*9` 格子大小箱子菜单中！

更多信息，请参阅 [MCIO Plugins](https://plugins.mcio.dev/docs/playermarket/intro) 中的插件文档。

## 功能特色

+ MySQL 数据库支持、跨服支持
+ 使用关系数据库的特性实现: 按条件检索、排序全球市场商品列表的功能
+ 自由度极高的界面配置，所有界面均可修改布局、文字、点击操作等等
+ 支持查看并管理玩家自己上架的商品列表
+ 玩家可选使用**命令**或**界面**上架商品到全球市场
+ 优秀的商品上架界面交互逻辑
+ 丰富的上架条件限制，阻止玩家上架那些你不想让他们上架的物品
+ 本地化支持，插件的几乎所有文字均可修改
+ 支持定期地、有条件地、有概率地以系统的名义自动上架商品，以便自动为玩家提供福利或稀有物品
+ 支持对玩家上架的物品进行分类
